📜 Script Reference Gravity Pulls Down
Scratch Blocks — on the BIRD sprite
when 🏁 clicked
go to x: (-180) y: (0)
set [y velocity] to (0)
forever
change [y velocity] by (-1)
change y by (y velocity)
if ‹(y position) < (-170)› then
set y to (-170)
set [y velocity] to (0)
💡 Tip Try "change [y velocity] by (-2)" for stronger gravity — the game gets much harder immediately!
🤔 Did You Know? Velocity accumulation under constant acceleration is Newtonian mechanics — F=ma in code. Every physics engine from Box2D to Unreal uses this exact loop every frame.

Pipes Scroll Left ScratchQuest • scratch.mit.edu
📜 Script Reference Pipes Scroll Left
Scratch Blocks — on the PIPE sprite
— PIPE: when I start as a clone →
show
repeat until ‹(x position) < (-240)›
change x by (-4)
delete this clone
💡 Tip Increase the scroll speed to -6 or -8 for a harder game. This one number controls overall game difficulty!
🤔 Did You Know? Parallax scrolling, endless runners, and side-scrollers all use this pattern: spawn off-screen right → scroll left → destroy off-screen left → repeat.
