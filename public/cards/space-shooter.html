Object Instantiation · Clone Pattern
Press SPACE to spawn a Ball clone that appears at the ship's position
Create the Clone ScratchQuest • scratch.mit.edu
📜 Script Reference Create the Clone
Scratch Blocks — on BALL & ROCKETSHIP
On BALL: when I start as a clone
go to [Rocketship] ← snaps to ship first
show
On ROCKETSHIP: when [space] key pressed
create clone of [Ball]
💡 Tip "go to [Rocketship]" must be the FIRST block in the clone script — before show. Otherwise the clone appears at the wrong position!
🤔 Did You Know? "create clone of [Ball]" is identical to writing "new Bullet()" in Java or Python. You are doing Object-Oriented Programming right now.

Repeat-Until · Destruction Pattern
The clone shoots upward and deletes itself when it leaves the screen
Fire the Laser ScratchQuest • scratch.mit.edu
📜 Script Reference Fire the Laser
Scratch Blocks — on the BALL sprite
On BALL: when I start as a clone
go to [Rocketship]
show
repeat until ‹(y position) > (175)›
change y by (15)
delete this clone
💡 Tip Increase "change y by (15)" to 20 or 25 for faster bullets. Faster bullets make the game easier!
🤔 Did You Know? "delete this clone" frees memory — just like garbage collection in Python and Java. Without it, hundreds of off-screen bullets would pile up.

Aliens appear randomly at the top and descend toward the ship
Spawn Alien Enemies ScratchQuest • scratch.mit.edu
📜 Script Reference Spawn Alien Enemies
Scratch Blocks — on the ALIEN sprite
ALIEN: when 🏁 clicked
hide
forever
wait (1.5) seconds
go to x: (pick random (-200) to (200)) y: (180)
create clone of [myself]
ALIEN: when I start as a clone
show
repeat until ‹(y position) < (-180)›
change y by (-3)
delete this clone
💡 Tip Lower the wait time (e.g. 0.8 secs) to spawn aliens faster. Higher "change y by" makes them fall quicker.
🤔 Did You Know? Two "when 🏁 clicked" scripts on one sprite run at the same time. In real code, this is called concurrency or async programming.

📜 Script Reference Detect the Hit
Scratch Blocks — on the ALIEN sprite
ALIEN: when I start as a clone
show
repeat until ‹(y position) < (-180)›
change y by (-3)
if ‹touching [Ball]?› then
play sound [pop] until done
delete this clone
💡 Tip Click the Sounds tab on the Alien sprite to add new sounds. "Zap", "Laser1", and "Pop" all feel great for hits.
🤔 Did You Know? "touching [Ball]?" polls for overlap 30 times per second. Game engines like Unity call this an AABB check — behind every game's hit detection.
