🖨️ How to Print the Cards

  1. Click a game above to open its card file
  2. Click the "Print These Cards" button at the top
  3. Game cards: set paper to 5" × 3.5" (or A4 "Fit to page")
  4. Action cards: set paper to 2.5" × 3.5" (playing card size)
  5. Enable double-sided printing → flip on short edge
  6. Each page alternates Front → Back so cards align when printed double-sided
  7. Cut along the card borders and laminate for durability!
