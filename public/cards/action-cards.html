Jump & Gravity Beginner
Motion · Variables
Script Reference Jump & Gravity
─ gravity (always runs) ─ when ⚑ clicked set [y power▼] to (0) forever change [y power▼] by (-1) change y by (y power) if <touching [ground▼]?> then set [y power▼] to (0) end end ─ jump (on space) ─ when [space▼] key pressed if <touching [ground▼]?> then set [y power▼] to (12) end
💡 Pro Tip
In Scratch, positive y = up. Jump sets y power = 12; gravity subtracts 1 each frame so the leap arcs back down.

Color Sense Advanced
Sensing
Script Reference Color Sense

Rollback Advanced
Motion
Script Reference Rollback
when ⚑ clicked forever set [prev_x▼] to (x pos) set [prev_y▼] to (y pos) move (5) steps if <touching color [■]?> go to x:(prev_x) y:(prev_y) end end
💡 Pro Tip
Save → Move → Check → Undo. A universal pattern for all collision detection!
